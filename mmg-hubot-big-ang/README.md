Hubot's Big Ang
===============


Description
-----------
Big Ang always knows what to say.


Dependencies
------------
Requires [Hubot](http://hubot.github.com/) of course.


Configuration
-------------
None.


Commands
--------
Hubot listens for key words, and replies with relevant "Big Ang" animated GIFs.
